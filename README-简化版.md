## 🚀 3 分钟快速开始

### 方法一：对话式安装（最简单）

**复制以下内容发给 OpenClaw：**

```
请帮我安装 OpenClaw 全域意图对齐协议：

1. 访问 GitHub：412984588/openclaw-alignment
2. 读取 USER.md, SOUL.md, AGENTS.md
3. 保存到 ~/.claude/ 目录
4. 完成后告诉我
```

**完成！** OpenClaw 会自动完成所有操作。

---

### 方法二：手动安装（适合熟悉文件操作的用户）

#### 步骤 1：下载文件

访问以下链接，右键"另存为"：

- [USER.md](https://raw.githubusercontent.com/412984588/openclaw-alignment/main/USER.md)
- [SOUL.md](https://raw.githubusercontent.com/412984588/openclaw-alignment/main/SOUL.md)
- [AGENTS.md](https://raw.githubusercontent.com/412984588/openclaw-alignment/main/AGENTS.md)

#### 步骤 2：复制到配置目录

**macOS 用户**：
1. Finder → 前往文件夹 → `~/.claude`
2. 粘贴下载的文件

**Windows 用户**：
1. 文件资源管理器 → 地址栏输入 `%USERPROFILE%\.claude`
2. 粘贴下载的文件

#### 步骤 3：验证

在 OpenClaw 中输入：
```
检查我的配置文件
```
