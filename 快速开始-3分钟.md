# 🎯 快速开始（3分钟搞定）

> 最简单的安装方式 - 只需要复制粘贴

---

## ✅ 安装核心文件（3个文件）

### 方法：在 OpenClaw 中直接配置

**复制以下内容，直接发给 OpenClaw：**

```
请帮我设置 OpenClaw 全域意图对齐配置：

1. 访问 GitHub 仓库：412984588/openclaw-alignment
2. 读取以下三个文件：
   - USER.md
   - SOUL.md
   - AGENTS.md
3. 将它们保存到我的 ~/.claude/ 目录
4. 完成后告诉我如何使用
```

**就这么简单！** OpenClaw 会自动完成下载和安装。

---

## 🎓 验证安装

安装完成后，输入：

```
检查一下我的 OpenClaw 配置是否正确
```

---

## 📖 核心文件说明

- **USER.md** - 你的个人偏好和使用习惯
- **SOUL.md** - AI 协作的基本规则和世界观
- **AGENTS.md** - 多 Agent 协作策略

如果你想修改配置，直接告诉 OpenClaw：

