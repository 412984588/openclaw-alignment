# 🖥️ 图形界面安装指南（适合非技术用户）

> 如果你不会使用命令行（Terminal/Git Bash），请按照以下步骤操作

---

## 📥 方法一：直接下载文件（最简单）

### 步骤 1：下载文件

访问 GitHub 仓库页面，直接点击下载：

**核心文件下载链接**：
- [USER.md](https://raw.githubusercontent.com/412984588/openclaw-alignment/main/USER.md) - 右键另存为
- [SOUL.md](https://raw.githubusercontent.com/412984588/openclaw-alignment/main/SOUL.md) - 右键另存为
- [AGENTS.md](https://raw.githubusercontent.com/412984588/openclaw-alignment/main/AGENTS.md) - 右键另存为

或者直接下载完整包：
- [下载 ZIP 压缩包](https://github.com/412984588/openclaw-alignment/archive/refs/heads/main.zip)

### 步骤 2：找到你的配置目录

#### macOS 用户

1. 打开 **Finder（访达）**
2. 按 `Command + Shift + G`（前往文件夹）
3. 输入：`~/.claude` 并回车
4. 如果没有这个文件夹，创建它：
   - 在 Finder 中，按 `Command + N` 新建文件夹
   - 命名为 `.claude`

#### Windows 用户

1. 打开 **文件资源管理器**
2. 在地址栏输入：`%USERPROFILE%\.claude` 并回车
3. 如果没有这个文件夹，创建它

### 步骤 3：复制文件

1. 解压下载的 ZIP 文件
2. 将以下文件复制到 `~/.claude/` 目录：
   - `USER.md`
   - `SOUL.md`
   - `AGENTS.md`

### 步骤 4：验证安装

在 OpenClaw 中输入：
```
检查 ~/.claude 目录下的文件
```

## 📋 方法二：使用 OpenClaw 内置功能（推荐）

### 步骤 1：在 OpenClaw 中创建配置文件

直接在 OpenClaw 对话中输入：

```
请帮我创建以下配置文件：

1. 在 ~/.claude/ 目录下创建 USER.md，内容为：
[我会提供内容]

2. 在 ~/.claude/ 目录下创建 SOUL.md，内容为：
[我会提供内容]

3. 在 ~/.claude/ 目录下创建 AGENTS.md，内容为：
[我会提供内容]
```

### 步骤 2：使用预设模板

让 OpenClaw 直接使用 GitHub 上的模板：

```
请访问 https://raw.githubusercontent.com/412984588/openclaw-alignment/main/USER.md
读取内容并保存到 ~/.claude/USER.md

然后依次创建 SOUL.md 和 AGENTS.md
```

---

## 🔧 方法三：使用一键配置（最简单）

### 在 OpenClaw 中执行以下命令：

```
请帮我执行以下操作：

1. 下载配置文件：
   - 从 GitHub: 412984588/openclaw-alignment
   - 读取 USER.md, SOUL.md, AGENTS.md
   - 保存到 ~/.claude/ 目录

2. 验证安装：
   - 检查文件是否存在
   - 确认内容正确

3. 配置完成提示：
   - 告诉我如何使用
```

---

## ✅ 验证安装成功

在 OpenClaw 中输入：

```
已安装 OpenClaw 全域意图对齐与自主进化协议
检查配置文件是否正确
```

如果安装成功，AI 会确认读取到以下文件：
- `~/.claude/USER.md`
- `~/.claude/SOUL.md`
- `~/.claude/AGENTS.md`
